# User Lifecycle

- User provides details (age, gender, height, weight, fitness experience)
- User confirms their fitness goals (bodybuild, powerlift, general strength training, etc. )
- User confirms their availabilty (how many days per week they can go to the gym, how many hours per visit, etc. )
- User performs baseline fitness test
- The app will output a weekly routine for user including:
    - Which days to go to the gym
    - Which workouts to do on which day
    - Which muscles are being targeted for each workout
    - How much weight to use
- After each workout, the user will update that session with:
    - Whether or not they went to the gym
    - Any substitutions they made for exercises (skipped/added exercises, increased/decreased weights, etc. )
    - Why they made any substitutions (injury, lack of time, too hard, etc. )
- With all of the post-workout information, the app will adjust this week's routine as needed and craft next week's routine

# Example

- User is a 24-year-old 6ft 180lb man
- User wants to improve general strength
- User is available on Tuesday, Thursday, Saturday, and Sunday
- User performs [insert baseline test when we figure that out]
- App will provide the following schedule:
    - Tuesday: Bench 3x(8x125lbs), Tri-extensions 3x(10x20lbs), Chest Flys 3x(8x35lbs), Shoulder Press 3x(10x30lbs)
    - Thursday: Cable Row 3x(8x35lbs), Pull-ups 3x10, Curls 3x(12x30lbs), Wrist-Curls 3x(12x45lbs)
    - Saturday: Bench 3x(8x130lbs), Shoulder Press 3x(10x30lbs), Cable Row 3x(8x35lbs), Curls 3x(12x30lbs),
    - Sunday: Squat 3x(8x155lbs), Quad Extensions 3x(10x60lbs), Hamstring Curls 3x(8x60lbs), Calf-Raises 3x(12x50lbs)
- User will complete the first routine on Tuesday
- After each exercise, the user will select the difficulty of that exercise (like nothing, felt easy, felt hard, couldn't finish)
- App will adjust the user profile accordingly (following workouts will be easier/harder/same)

# Other features

- User can connect fitness devices (smart watches, hr monitors, etc. ) to track data from workouts
- User can upload food eaten during the day to track calories and macros
- App will track calories burned during workouts and subtract from the User's daily calories
